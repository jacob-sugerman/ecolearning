# Wayback Machine Recovery URLs

Since the assets ARE available on the Wayback Machine, here are the specific URLs to check and download each asset manually.

## How to Use These URLs

1. Copy the Wayback Machine URL for each asset
2. Paste it into your browser
3. The Wayback Machine will show you available snapshots
4. Click on a snapshot date (usually from July-August 2022)
5. Download the file
6. Save it to the correct path in the `assets/` directory

## Quick Recovery Steps

1. **Try each Wayback URL** - Click the URLs above to see available snapshots
2. **Pick a snapshot date** - Look for dates around July-August 2022 when assets were uploaded
3. **Download each file** - Save to your computer
4. **Rename if needed** - Ensure filenames match exactly (see Local Path above)
5. **Place in assets/** - Copy files to the correct subdirectories
6. **Run update script**:
   ```bash
   python3 update_asset_paths.py
   ```
7. **Test** - Open index.html in a browser

## Notes

- The Wayback Machine URL format `https://web.archive.org/web/*/[URL]` shows all available snapshots
- Click on any blue/green circle in the calendar to access that snapshot
- For large files (like Hoh_Forest_Stream.mp4), the download might be slow
- Some assets may have multiple snapshots - any from 2022 should work
- If a file 404s, try removing the version parameter (`?v=...`) from the search

## After Recovery

Once you've downloaded all assets:

```bash
# Verify what you have
python3 update_asset_paths.py --dry-run
```
